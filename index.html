Projects

Vroom-Vroom

Navigating a maze, unconventionally.

My team built a Jenga car powered by servos and a Spark Core (basically a wireless arduino). By attaching an iPhone to stream video, we could create a 3D environment for a user on a computer using Oculus VR and Virtual Desktop. This user could remotely control the car using basic WASD controls. In addition, we added controllability via Myo Armband; different gestures signalled different movements. Recieved "Best Hardware Hack" at HoyaHacks 2016.

OmniTestr

Load testing websites with public API requests.

For PennApps, Sean Bae, Colin King, and I created a web app to load test any website (that you have permission to). Specifically, I worked on the master/slave interaction for the actual load testing, and data visualization with d3.js. While working on this, we also managed to win Hackenger, or the hackathon's huge puzzle.

DronePollock

Painting a canvas with a drone for the first time in history.

For Technica (all-ladies hackathon), a team of me and four others built software to get a drone to automatically splatter paint on a canvas. We used nodejs for the automation.

Startup Shell Alumni Map

Because community is the number one key.

The Startup Shell community is absolutely amazing, and it's important future batches stay in touch with those of the past. This map allows alumni to put themselves on a map as a resource for all current members. Built using javascript, Google Sheets, and Google Maps' API.

Cartly

Stop shopping and get back on task!

What started as a complete joke ended up as an actual chrome extension on the store. It uses Google Chrome's API and basic JavaScript to look for phrases that flag the page as a shopping page, and the program closes them.

SmartSpenders

Monitor family spending.

What started as a complete joke ended up asan actual chrome extension on the store. It uses Google Chrome's API and basic JavaScript to look for phrases that flag the page as a shopping page, and the program closes them. This was a finalist in the Daemon Dash Hackathon 2015.

CS Grade Checker

Because UMD CS doesn't tell me.

The university's cs department is phenomenal, but they never tell you when your grades are updated. So, I wrote a program to do so. It sends me a text everytime my grade changes. This was written entirely in python and is running off a web server.
